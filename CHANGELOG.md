# Changelog

All notable changes to the Mini Inventory Management System will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-23

### Added
- **Dashboard**: Real-time overview with key business metrics
- **Product Management**: Full CRUD operations with image support
- **Category Management**: Organize products into categories
- **Warehouse Management**: Track multiple warehouse locations
- **Supplier Management**: Manage supplier information and contacts
- **Customer Management**: Customer database with contact details
- **Invoice System**: Create and manage invoices with multiple items
- **Payment Tracking**: Track paid, partial, and unpaid invoices
- **Stock Monitoring**: 
  - Low stock alerts with customizable thresholds
  - Dead stock identification
  - Expired product tracking
- **Staff Management**: Employee records with role and salary tracking
- **Settings**:
  - Store name and branding configuration
  - Currency settings (30+ currencies supported)
  - Low stock alert thresholds
- **Export Features**:
  - Export to Excel
  - Export to PDF
  - Print-friendly views
- **Modern Dark Theme**: Professional UI with blue accent colors
- **Responsive Design**: Works on desktop and tablets
- **SQLite Database**: No external database required

### Technical Features
- Built with Streamlit 1.25+
- Python 3.9+ compatible
- Modular architecture for easy customization
- Service-based business logic layer
- Reusable UI components
