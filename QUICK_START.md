# 🚀 Quick Start Guide - Mini Inventory

## ⚡ Installation (3 Easy Steps)

### 1️⃣ Prerequisites
- **Python 3.9+** installed ([Download here](https://www.python.org/downloads/))
- Make sure to check **"Add Python to PATH"** during installation

### 2️⃣ Install
```bash
# Double-click this file:
setup.bat
```
Wait 1-2 minutes for installation to complete.

### 3️⃣ Run
```bash
# Double-click this file:
run.bat
```
The app will open automatically in your browser at `http://localhost:8501`

---

## 🎯 That's it! You're ready to go.

### 📚 First Steps:
1. **Dashboard** - View overview
2. **Settings** → **Store Settings** - Set your store name & logo
3. **Categories** → Add categories (e.g., Electronics, Furniture)
4. **Warehouses** → Add warehouses
5. **Products** → **Add Product** - Add your first product
6. **Invoices** → **Create Invoice** - Create your first invoice

## 🆘 Troubleshooting

### Python not found?
```bash
python --version
```
If this doesn't work, reinstall Python and check "Add to PATH"

### Dependencies not installing?
```bash
# Try manual installation:
pip install streamlit pandas openpyxl pillow reportlab
```

### Port 8501 already in use?
```bash
# Kill the process or change port:
streamlit run app.py --server.port 8502
```
